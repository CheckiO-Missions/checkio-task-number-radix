En esta misión recibirás una cadena de caracteres representando un número y su base. Tu tarea será convertir este número a formato decimal. La base del número estará comprendida entre 1 y 37. La cadena de caracteres contendrá dígitos y letras A-Z.

¡Ten mucho ojo con los casos donde el número no pueda ser convertido! Por ejemplo: "1A" no puede convertirse a base 9. En estos casos, tu función deberá devolver el valor -1.

Consejos: Este problema será muy sencillo de resolver si usas {% if "python 2" in interpreter|lower %} int() para convertir a entero y manejas las excepciones (Mira ValueError). {% else %} int() para convertir a entero y manejas las excepciones (Mira ValueError). {% endif %}

Entrada: Dos argumentos: un número (cadena de caracteres) y una base (entero).

Salida: El número convertido a formato decimal (entero).

Ejemplo: 

checkio("AF", 16) == 175
checkio("101", 2) == 5
checkio("101", 5) == 26
checkio("Z", 36) == 35
checkio("AB", 10) == -1

Utilidad: Aprender a hacer cambios de base entre diferentes sistemas numéricos y manejar excepciones.

Precondiciones:
re.match("\A[A-Z0-9]\Z", str_numero)
0 < len(str_numero) ≤ 10
2 ≤ base ≤ 36
